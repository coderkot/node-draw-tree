### Написать функцию для показа древовидной структуры.

Пример данных:

```
{
    "name": 1,
    "items": [{
        "name": 2, 
        "items": [
            { "name": 3 }, 
            { "name": 4 }
        ]}, 
        {
            "name": 5,
            "items": [{ "name": 6 }]
    }]
}
```
Пример запуска программы:

npm start
````
1
├── 2
│ └── 3
│ └── 4
└── 5
└── 6
````
